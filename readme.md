# @andrew-codes/color-functions
A variety of color manipulation functions written in JavaScript.

## Install
```bash
npm install @andrew-codes/color-functions
```

## Exposed Functions
- clamp : numeric
- toHex : string
- toHexString: string
- toHsla : hsla color object
- toHslaString: string
- toRgba : rbga color object
- toRgbaString: string
